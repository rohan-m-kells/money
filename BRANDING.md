# $ Money Core Branding Guidelines

## Logo
The official $ Money Core logo features the dollar sign ($) as the primary symbol, representing the project's focus on currency and financial systems.

## Official Name
- **Full Name**: $ Money Core
- **Short Name**: Money
- **Symbol**: $

## Color Scheme
- **Primary**: Green (#2F8D46) - representing growth and financial stability
- **Accent**: Gold (#FFD700) - representing value and prosperity
- **Neutral**: Dark Gray (#333333) - for text and backgrounds

## Typography
- **Logo Font**: Sans-serif with emphasis on the $ symbol
- **Documentation Font**: System default (Helvetica, Arial, or equivalent)

## Usage
- Always use the $ symbol before "Money Core" in official documentation
- When referring to the project in text, use "$ Money Core" for formal contexts and "Money" for informal contexts
- The dollar sign should be prominent in all visual branding

## Derived Works
This project is a fork of Bitcoin Core, maintaining compatibility with the MIT license while adding distinct branding centered on the $ symbol.
